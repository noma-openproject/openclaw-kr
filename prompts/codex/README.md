## 프롬프트 구조 (운영 규칙 섹션 8 기준)

모든 프롬프트는 4요소를 포함한다:

1. **목표**: 무엇을 달성할 것인지
2. **컨텍스트**: 어떤 파일/에러를 봐야 하는지
3. **제약조건**: 어떤 규칙을 따라야 하는지
4. **완료 조건**: 어떤 명령어가 통과해야 완료인지
