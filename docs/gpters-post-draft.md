# [오픈소스] Noma — 터미널 없이 AI 에이전트를 실행하는 한국형 데스크톱 앱

안녕하세요! 터미널 없이 AI 에이전트를 설치하고 실행할 수 있는 데스크톱 앱 **Noma**를 공유합니다.

## Noma가 뭔가요?

OpenClaw(오픈소스 AI 에이전트 플랫폼) 위에 **한국 사용자를 위한 레이어**를 얹은 데스크톱 앱입니다.

- 더블클릭으로 설치, 더블클릭으로 실행 (터미널 필요 없음)
- 카카오톡으로 외출 중에도 내 PC의 AI를 원격 제어
- 로또, KBO, 미세먼지, 택배조회 등 **한국 서비스 스킬 11개** 내장
- 보안 기본값 내장 (실행 전 확인, 검증된 스킬만)

## 왜 만들었나요?

ChatGPT를 쓰면서 "이걸 내 컴퓨터에서 돌리고 싶다"는 생각을 해보신 적 있나요?

OpenClaw가 그걸 가능하게 해주는데, 설치하고 설정하는 과정이 복잡합니다. Noma는 이 진입 장벽을 없앴습니다. 터미널 없이 .dmg/.exe 더블클릭만으로 시작할 수 있습니다.

핵심 차별점:
- **카카오톡이 리모컨** — 앱은 본체, 카카오톡은 리모컨. 밖에서도 내 PC의 AI에게 "로또 당첨번호 알려줘" 가능
- **한국 서비스 연동** — 로또, KBO, 미세먼지, 택배, SRT 시간표 등 한국에서 쓸모있는 스킬
- **비용 투명성** — 토큰 사용량, 모델별 비용을 실시간으로 보여줌

## 어떻게 쓰나요?

### 1단계: 다운로드
[GitHub Release 페이지](https://github.com/noma-openproject/openclaw-kr/releases/tag/v0.1.0-alpha.2)에서 macOS .dmg 또는 Windows .exe를 받으세요.

### 2단계: 설치
macOS: .dmg 열고 Applications에 드래그
Windows: 인스톨러 실행

### 3단계: 첫 실행
앱을 열면 온보딩 화면이 뜹니다. "시작하기"를 누르면 대시보드로 이동합니다.

### 4단계: AI 연결
ChatGPT 계정이 있다면 OAuth로 바로 연결됩니다. Gemini API 키도 사용 가능합니다.

## 스크린샷

(여기에 스크린샷 삽입)

1. 온보딩 화면 — 처음 실행하면 이런 화면이 뜹니다
2. 대시보드 — OpenClaw 대시보드가 데스크톱 앱 안에서 동작합니다
3. AI 대화 — 한국어로 자연스럽게 대화할 수 있습니다
4. 스킬 목록 — 설치된 스킬을 확인하고 관리합니다

## API 키 없이 바로 해볼 수 있는 것들

설치 후 아래를 대시보드에 입력해보세요:

- "이번 주 로또 당첨번호 알려줘"
- "오늘 KBO 경기 결과 알려줘"
- "서울 미세먼지 알려줘"

## 포함된 KR 스킬 (11개)

| 스킬 | 설명 | API 키 필요 |
|---|---|---|
| 로또 결과 | 당첨번호 조회 | 불필요 |
| KBO 경기 | 프로야구 결과 | 불필요 |
| 미세먼지 | 실시간 대기질 | 불필요 |
| 택배 배송조회 | CJ대한통운/우체국 | 불필요 |
| HWP 변환 | .hwp → 텍스트/마크다운 | 불필요 |
| 블루리본 맛집 | 맛집 검색 | 불필요 |
| 다이소 재고 | 상품/재고 검색 | 불필요 |
| SRT 시간표 | 열차 조회 | 불필요 |
| 서울 지하철 | 실시간 도착 | 서울시 API 키 |
| 우편번호 | 도로명주소 검색 | juso.go.kr 키 |
| 네이버 검색 | 뉴스/블로그 | 네이버 API 키 |

## 알려진 제한사항 (알파 버전)

- 첫 실행 시 gateway 시작에 최대 60초 소요될 수 있습니다
- Windows 빌드는 알파 테스트 중입니다 (문제 발견 시 GitHub Issues로 알려주세요)
- 메모리 기능은 현재 비활성화 상태입니다
