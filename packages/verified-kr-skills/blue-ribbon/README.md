# blue-ribbon

블루리본 서베이 근처 맛집 검색 OpenClaw 스킬.

## 사용법

```bash
node scripts/blue-ribbon.js <위치> [거리m] [개수]
node scripts/blue-ribbon.js 광화문
node scripts/blue-ribbon.js "37.573713, 126.978338" 2000 10
```
