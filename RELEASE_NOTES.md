# Noma v0.1.0-alpha

> 말이 아니라, 실행.

한국형 AI 에이전트 데스크톱 앱의 첫 알파 릴리스입니다.

## 설치 방법

### macOS
1. `.dmg` 파일을 다운로드합니다
2. 마운트 후 `Noma.app`을 Applications 폴더로 드래그합니다
3. OpenClaw가 설치되어 있어야 합니다 (아래 "사전 준비" 참고)

### Windows
1. `.exe` 인스톨러를 다운로드합니다
2. 실행하여 설치를 완료합니다
3. OpenClaw가 설치되어 있어야 합니다 (아래 "사전 준비" 참고)

## 사전 준비

```bash
# Node.js 22.16+ 필요 (24 권장)
npm install -g openclaw@2026.3.23-2
openclaw config set gateway.mode local
openclaw config set gateway.port 18789
```

## 첫 실행

```bash
# 터미널에서 gateway 시작
openclaw gateway --port 18789
```

그 후 Noma 앱을 실행하면 OpenClaw 대시보드가 데스크톱 창에 표시됩니다.

## 주요 기능

- **데스크톱 앱**: 터미널 없이 Electron 앱에서 AI 에이전트 실행
- **카카오톡 리모컨**: @noma-kr 채널로 외출 중에도 원격 제어
- **실행 영수증**: 토큰 사용량, 비용, 모델별 분포 실시간 추적
- **보안 기본값**: alpha-secure 프로필 (실행 전 확인, 검증된 스킬만)
- **Cheap Team**: Planner + Executor 자동 체이닝으로 비용 절감

## 알려진 제한사항

- OpenClaw gateway를 터미널에서 별도 실행해야 합니다 (Day 2에서 자동 시작 예정)
- Windows 빌드는 CI에서 생성되며 로컬 테스트가 제한적입니다
- 일부 KR 스킬은 외부 API 키가 필요합니다 (네이버 검색, 우편번호 등)
- 메모리 기능은 현재 비활성화 상태입니다 (OpenClaw 호환성 이슈)
